# STM32F0_DevBoard

STM32F030K6 Development Board Designed in Altium Designer
Offering :
  LCD 16x2 infercace
  74C922 Based Keypad Decoder
  Buzzer
  12V Relay
  EEPROM Memory
  Analog Input
  LM35 temperature Sensor
  Output LEDs
  CH340T Based USB to Serial
  Multiple input/output Voltage options
